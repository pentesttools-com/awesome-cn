    Awesome Arch

    Awesome Arch ¶

    > 一个很棒的项目的很棒的列表.

    Arch Linux 是独立开发的 x86-64 通用 GNU/Linux 发行版，用途广泛，足以适应任何角色. 开发侧重于简单性、极简主义和代码优雅. Arch 被安装为一个最小的基础系统，由用户配置，他们自己的理想环境通过只安装他们独特目的所需或期望的东西来组装.

    Arch-based projects¶

    Linux¶

    > 受 Arch Linux 强烈启发的 Linux 发行版.

    • Arch Linux 32 - 社区继续保持对 Arch Linux 的 32 位支持.
    • Arch Linux ARM - 用于 ARM 计算机的 Linux 发行版.

    Not Linux¶

    > 基于 Arch Linux 的非 Linux 项目.

    • PacBSD - 一个轻量级和灵活的 BSD 发行版，为使用 pacman 包管理器提供了一个基础系统.
    • Arch Hurd - Arch Linux 的衍生作品，将其移植到 GNU Hurd 系统，并使用针对 i686 架构优化的软件包.
    • ArchWSL - ArchLinux 作为 WSL 实例.

    Arch-based distros¶

    Desktop¶

    > 基于 Arch Linux 的通用 Linux 发行版.

    • Manjaro - 专业制作的基于 Linux 的操作系统，适合替代 Windows 或 macOS.
    • ArcoLinux - 功能齐全的发行版.
    • ArchBang Linux - 基于 Arch Linux 的轻量级发行版.
    • Archcraft OS - 一个极简主义的 Linux 发行版，专注于美学并基于 Arch Linux.
    • Archman - 基于 Arch Linux 的发行版，具有 Calamares 系统安装程序和预配置的桌面环境.
    • Bluestar Linux - 基于 Arch Linux 的 GNU/Linux 发行版.
    • Crystal Linux - Crystal Linux 是一个全新的基于 Arch Linux 的发行版. 完全适合初学者，易于使用且功能强大.
    • CachyOS - 一个基于 Arch Linux 构建的 GNU/Linux 发行版，它具有优化的 linux-cachyos 内核，利用先进的 BORE 调度程序实现无与伦比的性能.
    • EndeavourOS - 基于 Arch Linux 的滚动发布 Linux 发行版.
    • ArchLabs Linux - 基于 Arch Linux 的发行版，以 Openbox 窗口管理器为主要桌面界面.
    • Namib GNU/Linux - 基于（并兼容）Arch Linux 发行版的桌面操作系统.
    • ArchEX - 基于 Arch Linux 的全新 (190612) Linux live DVD. 座右铭是 KISS（保持简单愚蠢.
    • Ctlos Linux - 基于 Arch Linux 的发行版.
    • MagpieOS - 获得全新的 Arch 体验.
    • Netrunner Rolling - 基于 Manjaro/Arch Linux 的发行版.
    • PuppyRus-A - 基于 Arch、Puppy、Porteus 的轻量级、便携且灵活的 Linux.
    • TeLiSc OS - 一个基于 Arch Linux 的简单 GNU/Linux 发行版，为您提供带有 i3 窗口管理器的稳定 Arch Linux.
    • Qubes OS (Archlinux Template) - 一个以安全为中心的桌面操作系统，旨在通过隔离提供安全性（非官方 Archlinux 模板）.
    • XeroLinux - 一个维护良好的爱好者基于 Arch 的发行版，使用 ArcoLinux 的 ALCI 脚本构建，旨在为您提供稳定的基础，使安装 ArchLinux（使用 Calamares）和优化和定制的 KDE Plasma 变得更简单.

    Specific purpose¶

    > 基于具有特定用途的 Arch Linux 的 Linux 发行版.

    • BioArchLinux - （生物信息学/生物学） 生物学家 Arch 社区和生物软件 Arch 存储库.
    • BlackArch Linux - （安全/取证） - 基于 Arch Linux 的发行版，专为渗透测试人员和安全研究人员设计.
    • ArchStrike - (Security) - 基于 Arch Linux 的面向安全专家和研究人员的发行版.
    • SystemRescueCd - (Data Rescue) - An Arch-based Linux system on a bootable CD-ROM or USB drive, designed for repairing a system and data after a crash.
    • MorpheusArch Linux - (Data Rescue) - 基于 Arch Linux 的发行版.
    • LinHES - (MythTV) - 尝试使 GNU/Linux 和 MythTV 的安装尽可能简单.
    • UBOS - （服务器） - 一个 Linux 发行版，旨在大大减少设置和维护公共网络服务所需的时间.
    • TalkingArch - （辅助） - Arch Linux live ISO 映像的重新设计，修改为包括针对盲人和视障用户的语音和盲文输出.
    • BBQLinux - （Android 开发人员） - 为 Android 开发人员制作的用户友好的 Linux 发行版. 它具备构建 AOSP 或基于 AOSP 的发行版（如 LineageOS 或 OmniROM）的所有功能.
    • RaspArch - (Raspberry Pi/ARM) - Arch Linux ARM 的“重制版”.

    Pacman-based Linux distros¶

    > 使用 pacman 作为包管理器的 Linux 发行版.

    • KaOS - KaOS 是一个专注于 Qt 和 KDE 的独立发行版.

    SystemD-free Arch-based distros¶

    > 基于 Arch Linux 的 Linux 发行版，实现了 SystemD 的替代初始化.

    • Obarun - 基于 Arch Linux 的发行版，以 S6 init 软件代替 systemd.
    • Artix Linux - Arch-OpenRC 和 Manjaro-OpenRC 项目的分支（或作为自治项目的延续）.
    • Hyperbola GNU/Linux-libre - 一个社区驱动的努力，提供一个完全免费（如自由）的操作系统，该操作系统稳定、安全、简单、轻量级，试图通过长期支持（LTS）保持简单愚蠢（KISS）.
    • Parabola GNU/Linux-libre - Arch Linux 的非官方“自由”变体.
    • Alfheim Linux - 基于 Systemd 的免费 Arch Linux.

    Arch Linux Installers¶

    > 专门为方便安装 Linux Arch 而创建的安装程序.

    • Anarchy - 一个简单直观的 Arch Linux 安装程序.
    • Archfi - 在官方 Arch Linux 安装介质上启动后，只需一个简单的 bash 脚本向导即可安装 Arch Linux.
    • aui - Archlinux Ultimate Install 包括两个主要脚本，Lilo 和 Fifo，它们简化了 Archlinux Base 的安装过程，以及其他包、工具和图形用户界面.

    AUR Helpers¶

    Search and download¶

    • Auracle (C++) - Arch Linux 用户存储库的灵活命令行客户端.
    • pbget (Python) - 从 Git、ABS 和 makepkg 的 AUR 中检索 PKGBUILD 和本地源文件.
    • repoctl (Go) - 使管理本地 Arch Linux 存储库变得容易.
    • yaah (Bash) - Archlinux 用户存储库 (AUR) 的帮助程序.

    Search and build¶

    • Aurutils (Bash) - AUR 的辅助工具.
    • Bauerbill (Python) - 支持 AUR 和 ABS 的 Powerpill 扩展.
    • PKGBUILDer (Python) - Python 3 中的 AUR 助手（和库）.
    • repofish (Bash) - 用于管理本地 archlinux 存储库的简单 shell 脚本.
    • RUA (Rust) - 用于 Arch Linux 的构建工具，提供控制、审查和固定构建选项.

    Pacman wrappers¶

    • amethyst (Rust) - 为 Crystal 制作的快速、高效和轻量级的 AUR 帮助器和 Pacman 包装器，与任何基于 Arch 的 Linux 发行版兼容.
    • Aura (Haskell) - 用于 Arch Linux 和 AUR 的安全、多语言包管理器.
    • Pacaur (Bash) - 最小化用户交互的 AUR 助手.
    • Pakku (Nim) - 支持 AUR 的 pacman 包装器.
    • Paru (Rust) - 具有所有必需的现代包装器功能的 AUR 助手，由 yay 的前开发人员创建.
    • pikaur (Python) - 具有最小依赖性的 AUR 助手. 一次查看所有 PKGBUILD，然后在没有用户交互的情况下构建它们.
    • Trizen (Perl) - 轻量级 AUR 包管理器.
    • Yay (Go) - 另一个 Yogurt，一个用 Go 编写的 AUR Helper.

    Graphical¶

    • Argon (Python | GTK3) - Arch Linux 的轻量级 GUI 包管理器.
    • Cylon (Bash | TUI) - 一个 CLI TUI 菜单驱动的 bash shell 脚本，支持基于 Arch 的 Linux 发行版的更新、维护、备份和系统检查.
    • Pamac (Vala | GTK3) - Manjaro Linux 的图形包管理器，支持 Alpm、AUR、Appstream、Flatpak 和 Snap.
    • Pakku GUI (Python | GTK3) - 使用 pakku 的（安装|删除|升级）包的基本 GTK 接口.
    • PkgBrowser (Python | Qt5) - 用于浏览“pacman”数据库和 AUR 的实用程序.
    • Octopi (C++ | Qt5) - Arch Linux 包管理工具“pacman”的图形用户界面.
    • Yup (Go | TUI) - Arch Linux AUR Helper 具有 ncurses 功能和更好的搜索和排序.

    Maintenance¶

    • aur-out-of-date - 确定上游版本的过时 AUR 包.
    • aurpublish - Arch 用户存储库的 PKGBUILD 管理框架.
    • devtools - 用于构建和维护官方存储库包的 Arch Linux 发行版工具.
    • pkgbuild-watch - Alerts you to upstream activity.
    • pkgoutofdate - 检查 Arch 包是否过时.

    Other¶

    • arch-audit - 基于 Arch 安全团队数据的 pkg-audit 等实用程序.
    • aur-talk - 用于获取和显示 AUR 包注释的脚本.
    • aurvote-utils - 一组用于管理 AUR 投票的实用程序.
    • haskell-aur - 用于访问 Aurweb RPC 接口的 Haskell 库.
    • package-query - 用于查询 libalpm 和 AUR 的工具.
    • python3-aur - AUR 相关模块和辅助实用程序（aurploader、aurquery、aurtomatic）.
    • raur - 用于访问 Aurweb RPC 接口的 Rust 库.

    Other projects¶

    > 不属于任何先前类别的与 Arch Linux 相关或受其启发的项目.

    • Dotfiles Generator - 点文件生成器，允许在多个操作系统中快速配置不同的窗口管理器和工具，包括任何基于 Arch Linux 的发行版.
    • Termux Arch - 使用 Termux 在手机上运行 Arch Linux ARM.
    • archlinux/base - 基于 Arch Linux 的 docker 镜像.
    • Arch Linux on EC2 - 基于 Arch Linux 的 Amazon EC2 AMI
    • JuNest - 基于 Arch Linux 的轻量级发行版，无需 root 权限即可在任何 Linux 发行版上运行.
    • makepkg-optimize - pacman 的补充 tidy、buildenv 和可执行脚本的集合，它为 build() 和 package() 阶段的多种优化提供宏.

    Inactive projects¶

    > 属于列表一部分但已停止或很长时间未收到更新的项目.

    • Arch XFerience - 基于 Arch Linux 的现代操作系统，具有大量软件包和功能，专为日常操作而构建. （无效）
    • Chakra GNU/Linux - 一个用户友好且功能强大的发行版和 live CD，最初是从 Arch Linux 派生出来的. (discontinued)
    • Archphile - (Raspberry Pi/ARM/Audio) - 另一个用于嵌入式板的 linux 音频发行版. （无效）
