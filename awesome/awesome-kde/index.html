    Awesome kde

    Awesome KDE ¶

    > 用于 KDE 桌面环境的 awesome 应用程序、扩展、模块、主题和工具的精选列表.

    KDE 是一个免费软件社区，致力于创造开放和用户友好的计算体验，提供先进的图形桌面、各种用于通信、工作、教育和娱乐的应用程序以及一个可以轻松构建新应用程序的平台.

    > 标有（图钉）的项目是托管在 KDE 自己的基础架构上并由 KDE 社区开发的项目.

    Applications¶

    提示：最常用的 KDE 社区应用程序的概述列在 official website.

    Internet & Networking¶

    • Akregator - 使您能够关注新闻站点、博客和其他支持 RSS/Atom 的网站.
    • Falkon - 基于 QtWebEngine 的跨平台网络浏览器.
    • Firefox KDE - 带有 OpenSUSE 补丁的 mozilla.org 独立网络浏览器，与 KDE 更好地集成.
    • GSConnect - KDE Con​​nect 的完整实现，特别是针对 GNOME Shell 与 Nautilus、Chrome 和 Firefox 的集成.
    • KDE Connect - 跨所有设备进行通信的项目：在台式计算机上接收电话通知，通过手机控制桌面等.
    • KGet - 多功能且用户友好的下载管理器.
    • KMail - KMail 是 Kontact 的电子邮件组件，Kontact 是 KDE 的集成个人信息管理器.
    • KNetworkManager - 用于后端 NetworkManager 软件的 KDE 客户端.
    • Konqueror - 使用 KHTML 渲染引擎（和可选的 Webkit），因此它支持最新的网络标准，如 HTML5、JavaScript、CSS3 等.
    • Konversation - 基于 KDE 平台构建的用户友好的 Internet 中继聊天 (IRC) 客户端.
    • KRDC - 一个客户端应用程序，允许您在另一台运行兼容服务器（VNC 和 RDP）的计算机上查看或控制桌面会话.
    • KTorrent - 一个 BitTorrent 应用程序，允许您使用 BitTorrent 协议下载文件.
    • Otter Browser - Otter Browser 旨在使用 Qt5 重现经典 Opera (12.x) UI 的最佳方面.
    • Plasma-browser-integration - 一套新功能，使 Firefox, Chrome 基于 Chromium 的浏览器可与您的桌面一起使用.
    • qBittorrent - 基于 Qt 工具包和 libtorrent-rasterbar，用 C++ 编程的高级 BitTorrent 客户端.
    • Qt WebBrowser - Qt WebBrowser 是使用 Qt 和 Qt WebEngine 的功能开发的嵌入式设备浏览器.
    • Smb4K - 高级网上邻居浏览器和 Samba 共享安装实用程序.

    Graphics & Imaging¶

    • digiKam - 具有开源功能的专业照片管理.
    • Gwenview - KDE 提供的快速且易于使用的图像查看器.
    • KColorChooser - 从屏幕或调色板中选择颜色的简单应用程序.
    • KolourPaint - 一个简单的绘画程序，可快速创建光栅图像. 它可用作润色工具和简单的图像编辑任务.
    • Krita - 专为概念艺术家、插画师、遮罩和纹理艺术家以及 VFX 行业设计的绘画工具.
    • Okular - 通用文档查看器.
    • Spectacle - 用于捕获桌面屏幕截图的应用程序.

    Multimedia¶

    • Amarok - Amarok 是一款功能强大的音乐播放器，适用于 Linux、Unix 和 Windows，界面直观.
    • Elisa - 由 KDE 社区开发的音乐播放器，力求简单易用.
    • Haruna - 使用 Qt/QML 和 libmpv 构建的开源视频播放器.
    • Kdenlive - 使用 MLT 视频框架的 Linux 非线性视频编辑器.
    • KMix - 声道混音器和音量控制.
    • KWave - 基于 KDE Frameworks 5 构建的声音编辑器.
    • SMPlayer - 具有内置编解码器的媒体播放器，几乎可以播放所有视频和音频格式.
    • VLC - 一个免费和开源的跨平台多媒体播放器和框架，可以播放大多数多媒体文件和各种流媒体协议.

    Office & Productivity¶

    • Calligra - 包括高效且功能强大的办公组件.
    • Kate - KDE 的多文档、多视图文本编辑器.
    • Kronometer KDE 的秒表应用程序.
    • KWrite - KDE 的文本编辑器，基于 KDE Frameworks 编辑器组件.
    • RSI Break - RSIBreak 可以用来提醒你不时休息一下.

    System Administration¶

    • Discover - 适用于终端不是安装应用程序和附加组件或更新的最佳解决方案的用户和发行版.
    • Dolphin - 包含在 KDE 桌面中的文件管理器.
    • KDiskFree - 显示可用的文件设备（硬盘分区、驱动器等）及其容量、可用空间、类型和安装点等信息.
    • KInfoCenter - 提供有关计算机系统的详细信息的实用程序.
    • Konsole - KDE 桌面中包含的终端仿真器.
    • Krusader - 用于 KDE 的高级双面板（命令式）文件管理器.
    • KSystemLog - 系统日志查看器工具.
    • Kup - KDE 等离子桌面的备份调度程序.
    • Sweeper - Sweeper 有助于清除用户在系统上留下的不需要的痕迹，并通过删除未使用的临时文件重新获得磁盘空间.
    • Yakuake - 基于 KDE 的 Konsole 技术的下拉式终端仿真器.

    Utilities & Tools¶

    • Android File Transfer - 具有简约 UI 的可靠 MTP 客户端.
    • Ark - 归档工具.
    • fancontrol-gui - 风扇控制的 GUI，它是 lm_sensors 的一部分.
    • KDE Wallet Manager - 管理 KDE 系统上的密码. KDE 钱包子系统允许对与 KDE 钱包集成的每个应用程序进行访问和密码管理.
    • KeePassXC - Keepass 密码管理器的跨平台社区驱动端口.
    • KFind - 一个独立的搜索工具，由 KRunner 启动或从您的菜单中启动. 它还作为“工具”菜单中的“查找文件”集成到 Konqueror 中.
    • KRename - 重命名多个文件的实用程序.
    • KWin-lowlatency - 尝试减少 KDE 中使用的流行 KWin 合成器的延迟和卡顿.
    • KWinFT - Wayland 合成器和 X11 窗口管理器.
    • QtPass - 多平台 GUI pass.
    • Wacom GUI - 用于 KDE 的 Python/PyQt Wacom GUI.
    • Yin-Yang - Linux 的自动夜间模式.

    Development & Programming¶

    • KDevelop - 适用于 C、C++、Python、QML/JavaScript 和 PHP 的跨平台 IDE.
    • Kompare - 差异/补丁前端.
    • Okteta - 十六进制编辑器.
    • Qt Designer - 用于使用 Qt Widgets 设计和构建图形用户界面 (GUI) 的 Qt 工具.
    • Umbrello - UML Modeller 是一个基于 KDE 技术的统一建模语言 (UML) 图表程序.

    Education¶

    • KTurtle - 一个教育编程环境，旨在让学习如何编程变得尽可能简单.
    • LabPlot - 交互式绘图和科学数据分析.

    Customization¶

    注意：定制很快就会过时，建议查看 KDE Store 或使用 Discover 反而.

    Plasma desktop¶

    Extensions¶

    • Bismuth - 一个 KDE Plasma 扩展，让您可以自动平铺窗口并通过键盘管理它们，就像在经典的平铺窗口管理器（i3、dwm 或 XMondad）中一样.
    • Dynamic Wallpaper Engine - 一个壁纸插件，可以根据您所在位置的当前时间不断更新桌面背景.
    • Hoppla - 用于控制 Philips Hue 灯的 KDE Plasma 桌面小部件.
    • Kvantum - 基于 SVG 的 Qt4/Qt5、KDE ​​和 LXQt 主题引擎，强调优雅、易用性和实用性.
    • Latte-Dock - Latte 是基于 Plasma 框架的扩展坞，可为您的任务和 plasmoid 提供优雅而直观的体验.
    • Places Widget - 允许访问用户位置的 Plasma 5 小部件.
    • Window AppMenu Applet - 等离子 5 小程序以显示窗口应用程序菜单.
    • Window Title Applet - 在面板中显示当前窗口标题和图标的 Plasma 5 小程序.

    KCM Modules and Systems¶

    • kcm-wacomtablet - Wacom Linux 驱动程序的 KDE GUI.
    • kde-gtk-config - KDE 的 GTK2 和 GTK3 配置器.
    • kded_rotation - 一个小型的、hacky KDED 模块，用于处理平板电脑和混合笔记本电脑上的自动屏幕旋转.

    Themes¶

    • Adapta KDE - 流行的 GTK 主题 Adapta for Plasma 5 桌面的一个端口，带有一些添加和附加功能.
    • Arc KDE - 流行的 GTK 主题 Arc for Plasma 5 桌面的一个端口，带有一些添加和附加功能.
    • Breath - Manjaro Linux (Plasma 5) 的呼吸主题.
    • Canta - KDE Plasma 桌面的平面设计主题.
    • ChromeOS - KDE Plasma 桌面的材料设计主题.
    • Darkine KDE - KDE Plasma 5 桌面、SDDM 等主题.
    • Fluent - 受微软启发的 KDE Plasma 桌面流畅设计主题.
    • Helium - Plasma 5 的半透明主题.
    • Hello - 各种配色方案、主题、图标等的汇编，适用于 KDE Plasma 桌面. 它没有意见，让你负责.
    • Layan - KDE Plasma 桌面的平面设计主题.
    • Materia KDE - 流行的 GTK 主题 Materia 的端口，用于 Plasma 5 桌面，带有一些添加和附加功能.
    • McMojave - 适用于 KDE Plasma 桌面的类似 MacOSX Mojave 的主题.
    • Orchis - KDE Plasma 桌面的材料设计主题.
    • Qogir - KDE Plasma 桌面的平面设计主题.
    • WhiteSur - KDE Plasma 的 MacOS Big Sur 主题.

    Qt and GTK+ Applications Appearance¶

    • Akava-Kv - KDE 的面向用户的 Kvantum 主题.
    • breeze-gtk - 为匹配 KDE 的 Breeze 而构建的 GTK 主题.
    • Breeze10 - KDE Plasma 的 Windows 10 风格窗户装饰.
    • BreezeBlurred - 用 Qt C++ 编写的 KDE Breeze 窗口装饰的分支.
    • BreezeEnhanced - 带有附加选项的 KDE Breeze 装饰分支.
    • Chrome-KDE5BreezeDark - 使用 Breeze Dark 窗口装饰时，KDE5 上的 Chrome 主题.
    • KDE-Rounded-Corners - Rounds the corners of your windows.
    • SierraBreeze - 用 C++ 编写的 KDE Plasma 类 OSX 窗口装饰.
    • Sweet KDE - KDE 等离子的黑暗和现代主题.
    • XBoomer - KDE Plasma 的 XP 窗口装饰.

    Icons¶

    • Antu - 为等离子桌面设计的流畅图标主题.
    • Candy - 甜蜜的渐变图标.
    • Flatery - 平面风格的 Linux 图标主题.
    • La Capitaine - 图标包的灵感来自最新版的 macOS 和 Google 的 Material Design.
    • Papirus - Material 图标主题，最初基于 Paper.
    • Tela - 平面彩色设计图标主题.
    • Zafiro - 使用平面设计技术创建的极简主义图标，使用褪色的颜色并始终伴随着白色. 优先考虑的是简单.

    Cursors¶

    • Bibata - 基于材质的光标主题.
    • Capitaine - 受 macOS 启发并基于 KDE Breeze 的 x 光标主题.

    Scripts¶

    • Grid-Tiling-Kwin - 自动平铺窗口的 kwin 脚本.
    • Kröhnkite - KWin 的动态平铺扩展.
    • kwin-tiling - kwin 的平铺脚本.

    Display Manager¶

    • Chili - KDE Plasma 5 最热门的登录主题.
    • Delicious - 适合 DE 收藏家/爱好者的美味 SDDM 主题.
    • Sugar Candy - 可用于 SDDM 显示管理器的最甜蜜的登录主题.

    Documentation and Resources¶

    • KDE Wikis - KDE 维基是 KDE 桌面、应用程序、团队、事件等信息和文档的来源.
    • KDE Invent - 发现 KDE 项目、组和片段.
    • KDE Bugtracker - KDE 的官方 Bugtracker.
    • ArchWiki - Arch Linux KDE 维基百科.
    • Gentoo Wiki - Gentoo KDE 维基百科.
    • KDE Configs - KDE 配置文件列表等.

    Articles¶

    • KDE.news - 官方 KDE 新闻网站.
    • Planet KDE - KDE 贡献者的博客.
    • Phoronix - Phoronix KDE 新闻档案.
    • Nate Graham's Blog - Linux 和 KDE 历险记.
    • Nico's Blog - 从事 KDE 工作的德国程序员.
    • Kai Uwe's Blog - Plasma 开发者的故事.

    Communities¶

    • Kreddit - Reddit 上的 KDE 社区.
    • KDE Store - 供开发人员共享 Plasma 定制的官方 KDE 商店.
    • Stack Overflow - 标有 kde 的问题.
