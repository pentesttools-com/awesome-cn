    Awesome Android¶

    About¶

    A curated list of awesome Android libraries 和 resources . 对于一般的 Java 库，请查看 awesome-java.

    Understand How Your Android App is Doing with Real-Time Contextual Insights From Your Users¶

    >Instabug 帮助 Android 开发人员和产品团队轻松收集错误和来自 Beta 测试人员和用户的反馈，以更快地迭代并交付高质量的应用程序. Instabug 会自动附加屏幕截图、设备详细信息、网络日志、重现步骤以及解决问题和确定产品待办事项优先级所需的大量其他重要见解. > 移动团队通过与 Github、Jira、Slack、Zendesk 等第三方工具无缝集成来加速他们的工作流程. Instabug is offering Awesome Android community an exclusive 15% discount on all paid plans. Get Started!

    How to Use¶

    Awesome-Android 是一个令人惊叹的列表，适用于需要在其应用程序中添加特定功能的人，因此最好的使用方法是： - 只需按 command + F 即可搜索关键字 - 浏览我们的内容菜单

    Emulators¶

    • AndY
    • ARChon
    • BlueStacks
    • Genymotion
    • nox

    Libraries¶

    Charts¶

    • AChartEngine - 图表引擎.
    • EazeGraph - 图表和图形库.
    • WilliamChart - 具有良好动态功能的图表库.
    • HelloCharts - 支持缩放、滚动和动画的图表和图形库.
    • MPAndroidChart - 支持手势缩放和拖动的 Android 图表和图形库.
    • ArcChartView - 绘制创意统计弧形图.
    • AnyChart - 数据可视化库，交互式图表.

    Cloud Services¶

    • CloudRail - 统一的 API 库：云存储、社交资料、支付、电子邮件、SMS 和 POI.

    Data binding¶

    • Anvil - 受 React 启发，用于创建反应式 UI 组件的小型库. 提供数据绑定和事件监听器绑定，非常适合MVVM.
    • Data Binding Library - 官方 Android 数据绑定库，用于编写声明式布局并最大限度地减少绑定应用程序逻辑和布局所需的胶水代码.

    Dependency Injection¶

    • Dagger 2 - 适用于 Android 和 Java 的快速依赖注入器.
    • Butter Knife - 查看 Android 的“注入”库.
    • ActivityStarter - Android 库提供更简单的方法来启动具有多个参数的活动.
    • AndroidAnnotations - 在编译时使用依赖注入的 Java 注释.
    • Toothpick - 基于作用域树的 Java 依赖注入 (DI) 库.

    Android Services¶

    • Remoter - 使用纯 Java 接口的 Android 远程 IPC 服务的 Android AIDL 替代方案.
    • Service Connector - 将 Android 服务和回调绑定到字段和方法.

    Game Development¶

    • Libgdx - 跨平台游戏引擎和SDK. Open Source
    • Vuforia - 增强现实图书馆.
    • Unity - 跨平台游戏创建系统.
    • Rajawali - Android OpenGL ES 2.0/3.0 引擎
    • Cocos2d-x - 跨平台 2d 游戏框架.
    • JustWeEngine - 一个简单的开源 Android 原生游戏框架.

    Security¶

    • libsignal-protocol-java - 在同步和异步消息传递环境中工作的棘轮前向保密协议.
    • Themis - 使典型加密方案易于使用的多语言框架：静态数据、经过身份验证的数据交换、传输保护、身份验证等.

    GUI¶

    • Pull to refresh - v4 支持库中提供了滑动刷新布局.
    • Cardslib - 用于构建 UI 卡的 Android 库.
    • AndroidStaggeredGrid - 网格视图，支持具有不同大小的行的多列.
    • Flow - 有助于将应用程序描述为适度独立屏幕的集合的库.
    • SortableTableView - 一个包含简单 TableView 和高级 SortableTableView 的 Android 库，提供大量定制可能性以满足所有需求.
    • MaterialProgressBar - 具有一致外观的材质设计 ProgressBar.
    • AndroidFillableLoaders - 使用 SVG 路径的可填充进度视图. 创建有趣的应用程序徽标也是不错的选择.
    • NexusDialog - 允许您使用很少的代码在 Android 中轻松快速地创建表单.
    • Snap RecyclerView Utils - 在不创建适配器的情况下填充单个或多个布局 RecyclerView.
    • MultiSnapRecyclerView - 用于多次捕捉 RecyclerView 的 Android 库
    • SwipeableCard - 实现像街景一样的刷卡！！
    • ElasticProgressBar - 漂亮的载入栏.
    • EntryScreenManager -介绍/进入/演练/开始屏幕.
    • EasyIntro - 适用于您的 Android 项目的灵活、易于使用的多合一应用程序介绍库.
    • Material-Calendar-View - 与 API 8+ 兼容的 Material Design 日历
    • CrunchyCalendar - 具有无限滚动、日期范围选择和颜色自定义的材料日历小部件.
    • SmoothOverscrollableScrollView - 具有平滑滚动的小型自定义视图. 您可以添加带有比例背景的标题
    • SectionedRecyclerViewAdapter - 允许将 RecyclerView 拆分为带有页眉和/或页脚的部分的适配器.
    • DragListView - 拖放以重新排序列表、网格或面板中的项目.
    • Animated Expanding ListView - 动画扩展 ListView 在扩展或折叠列表视图项目的内容时提供精美的动画.
    • TastyToast - 带有图标和颜色的祝酒词.
    • DotLoader - 带点的可自定义加载动画.
    • PodSlider - 符合材料设计规范的可定制滑块小部件.
    • TapTargetView - 用于功能发现的 Material Design 指南中的点击目标的实现.
    • ShowCaseView - ShowcaseView 库旨在通过有吸引力的平面覆盖向用户突出显示和展示应用程序的特定部分.
    • MaterialIntroScreen - 具有易于扩展的 API 的 Material Intro Screen 实现.
    • FloatingView - FloatingView 可以让目标视图漂浮在锚点视图上方，动画很酷.
    • Timecon - 易于使用的动画时钟图标
    • Audiogram - 轻量级音频进度条
    • Bubbles for Android - Facebook 喜欢聊天气泡库
    • Litho (By Facebook) - 用于在 Android 上构建高效 UI 的声明性框架.
    • MultiViewAdapter - Recyclerview 适配器库，用于创建可组合的视图持有者.
    • LGSnackbar - 本机 Android Snackbar 的易于使用和可定制的包装器，它在多个活动中保持可见.
    • ShimmerLayout - Android 应用程序的内存高效闪烁效果.
    • CircleProgressBar - 一个用于为 Android 创建循环进度条的简单库.
    • Easy-Signature-Android - 一个简单的 ui 库，提供可插入的签名视图.
    • Flashbar - 一个高度可定制、功能强大且易于使用的 Android 警报库.
    • YuanaItemSettingView - 适用于 Android 的可自定义项目设置视图.
    • Gradients - 精选的精美渐变系列.
    • OneAdapter - 具有多个模块和挂钩的 RecyclerView 适配器可简化和增强使用，同时防止常见错误.

    Paginate¶

    • NoPaginate - 简单的安卓分页库

    ActionBar¶

    • ActionBarSherlock - 适用于旧版 Android 的 ActionBar.
    • FadingActionBar - 在新的 Play Music 应用程序中可以看到渐变操作栏效果.

    Navigation¶

    • SlidingMenu - 用于创建带有滑入式菜单的应用程序的库.
    • SlidingTutorial - 有助于创建很棒的滑动 android 应用程序教程的简单库.
    • PagerSlidingTabStrip - 在 ViewPager 的不同页面之间导航的交互式指示器.
    • Page View indicator - 支持水平滚动 ViewPager.
    • RecyclerTabLayout - 使用 RecyclerView 实现的高效 TabLayout 库.
    • MaterialDrawer - 简单采用材料设计导航抽屉.
    • Debug-Artist - 调试菜单可轻松启用 leakcanary、scalpel 和其他工具.
    • Floating-Navigation-View - 一个简单的浮动操作按钮，显示锚定的导航视图.

    Animations¶

    • Rebound - Rebound 是一个模拟弹簧动力学的 Java 库.
    • Android View Animations - 可爱的视图动画集合.
    • Android-Transition - 允许轻松创建对用户输入做出反应的视图转换.
    • Android-View-Actions - 使为视图创建复杂的动画变得容易.
    • Swipper - 用于控制音量、亮度和搜索的滑动手势的 Android 库.
    • Spotlight - 为教程或演练等点亮项目的 Android 库...

    Images¶

    • Crescento - 通过在图像视图下方添加曲线来探索材料设计的新风格.
    • android-crop - 用于裁剪图像的库项目.
    • CircularImageView - 圆形图像的自定义视图，同时保持最佳绘制性能.
    • Android-Image-Filter - 用于轻松应用图像过滤器的库项目.
    • Compressor - Compressor 是一个轻量级且功能强大的安卓图像压缩库.
    • ShapeImageView - 以不同形状显示图像的库.

    Inputs¶

    • FloatingLabel - FloatingLabel 允许您创建一种打击类型的 EditText. 没有 Gradle 或 Maven 支持.
    • MaterialEditText - 支持浮动标签、单行省略号、最大/最小字符、帮助文本和带有自定义颜色的错误文本.
    • EmojiCompat - 将表情符号添加到您的应用程序
    • MaterialSearchBar - 适用于 Android 的 Material Design 搜索栏
    • InputMask - 基于模式的用户输入格式化程序、解析器和验证器.
    • SweetPassword - 允许自定义切换按钮的密码 EditText
    • VoiceOverlay - 一个叠加层，可获得用户的语音许可并在可自定义的 UI 中以文本形式输入.

    View Pagers¶

    • Material Dots Indicators - 用于查看寻呼机的三种材料点指示器样式.

    Loading Images¶

    • Picasso - 一个强大的安卓图像下载和缓存库.
    • Universal Image Loader - Asynchronous, out of the box loading and caching of images.
    • Glide - 适用于 Android 的图像加载和缓存库，专注于平滑滚动，由 Google 推荐.
    • Fresco - 用于管理图像及其使用的内存的 Android 库.
