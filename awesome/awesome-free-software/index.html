    免费软件

    Awesome Free Software ¶

    此列表中的软件是根据允许任何人出于任何目的使用、修改和共享该软件的条款分发的. 两个最流行的许可证是 MIT License 和 GNU GPL， 但是这里有 many other licenses 授予这些权限.

    Software¶

    Audio¶

    • Audacity - Sound recording and post-processing. (GNU GPLv2+)
    • Ardour - 数字音频工作站（DAW）. （GNU GPLv2)
    • Libre.fm - 免费串流、下载、混音和分享音乐. （GNU AGPLv3)
    • LMMS - 完整的数字音频工作站. （GNU GPLv2+)
    • mStream - 用于跨多个设备同步和流式传输音乐的软件套件. （GNU GPLv3)
    • Nuclear Music Player - 流媒体音乐播放器，自动从免费来源查找音乐. （GNU AGPLv3)

    Command Line Tools¶

    • asciinema - 终端会话记录器. （GNU GPLv3)
    • Clipboard - 随时随地通过命令行剪切、复制和粘贴任何内容. （GNU GPLv3)
    • Cygwin - 适用于 Windows 的类 Unix 环境，提供对 Linux 系统上常见的许多工具的访问. （GNU GPLv3)
    • Logdissect - 用于分析日志文件和其他数据的 CLI 实用程序和 Python API. （MIT)
    • PuTTY - 通过 SSH、Telnet、SCP 和 rlogin 协议以及原始套接字连接进行连接的远程终端仿真器. （MIT)
    • ranger - 适用于 GNU/Linux 的最小、轻量且美观的文件浏览器. 它在终端窗口中运行，因此非常适合 i3 等窗口管理器. vi 键绑定允许执行快速操作，包括重命名和删除. （GNU GPLv3)
    • GNU stow - GNU Stow 是一个符号链接场管理器，它采用位于文件系统上不同目录中的不同软件和/或数据包，并使它们看起来安装在同一位置. （GNU GPLv3)
    • SimpleLocalize - 用于管理软件项目中的 i18n 密钥的开源工具.

    Communication¶

    • Conversations - 适用于 Android 的 XMPP/Jabber 客户端. （GNU GPLv3)
    • Gajim - 桌面版 XMPP/Jabber 客户端. （GNU GPLv3)
    • Jitsi Meet - 使用 WebRTC 保护视频会议和桌面共享. （Apache License 2.0)
    • Mail-in-a-Box - 易于部署的云计算机邮件服务器包. （CC0)
    • Matrix - 分散、持久和可互操作通信的开放标准. （Apache License 2.0)
    • Mattermost - Slack 的开源私有云替代方案. （Licensing)
    • ProtonMail - 基于网络的安全电子邮件服务. （MIT)
    • Ring - 通过分布式服务进行电话、电话会议和媒体共享. （GNU GPLv3)
    • Signal - 适用于桌面、Android 和 iOS 的私人消息传递和通话. （GNU GPLv3)
    • Tox - 具有语音、视频和屏幕共享功能的加密分布式消息传递平台. （GNU GPLv3)
    • Tutanota - 免费加密电子邮件，支持企业帐户. （GNU GPLv3)

    File Management¶

    • 7-Zip - 用于创建和打开压缩文件的文件存档器. （GNU LGPLv2.1+)
    • Filestash - 类似 Dropbox 的 Web 客户端，用户可以自带自己的后端（FTP、SFTP、Webdav、S3、Minio 等）. （GNU AGPLv3)
    • FileZilla - 通用 FTP 解决方案. （GNU GPLv2+)
    • WinSCP - 适用于 Windows 的 SFTP 和 FTP 客户端（GNU GPLv3)

    File Sharing¶

    • qBittorrent - 易于使用的 BitTorrent 客户端. （GNU GPLv2+)
    • Sharry - 方便的文件共享 Web 应用程序，允许您上传文件并获取可共享的 URL. （GNU GPLv3+)

    Graphics¶

    • Blender - 用于建模、模拟和动画的 3D 创建套件. （GNU GPLv2+)
    • ExifCleaner - GUI 应用程序可通过简单的拖放从图像和视频中删除 exif 元数据. （MIT)
    • GIMP - 图像处理软件. （GNU GPLv3)
    • Inkscape - 适用于所有平台的专业矢量图形编辑器. （GNU GPL)
    • Krita - 艺术家制作的绘画程序. （GNU GPLv3)
    • Pinta - Gtk# clone of Paint.NET. (MIT)
    • Piskel - 在线精灵编辑器. （Apache License 2.0)

    IDEs¶

    • Eclipse - 可扩展的 IDE，支持最新的 Java 版本以及其他语言. （EPL)
    • IntelliJ IDEA Community Edition - Java/Android IDE，目标是最大限度地提高开发人员的生产力. （Apache License 2.0)

    Office¶

    • LibreOffice - 带有文字处理程序、电子表格和演示程序的办公套件. （MPLv2)
    • Open Office - 提供基于开放标准的全功能办公生产力套件. （Apache License 2.0)
    • Zathura - 高度可定制且功能齐全的文档查看器. 它提供了简约且节省空间的界面以及主要侧重于键盘交互的简单使用. （zlib License)

    Operating Systems¶

    • Debian - 最早的类 UNIX 操作系统之一，致力于将非自由软件排除在其系统之外. （DFSG)
    • Devuan - 没有 systemd 的 Debian 分支. （Licenses)
    • PureOS - FSF 认可 Debian 的衍生版本. （多个许可证）
    • LineageOS - 适用于手机和平板电脑的 Android 发行版以及 CyanogenMod 的后继者. （多个许可证）

    Other¶

    • Disroot - 此列表中许多服务的托管版本. （多个许可证）
    • OpenMapTiles - 用于自行托管 50 多种语言的 OpenStreetMap 地图的开源工具集. 它提供栅格和矢量切片、GIS 程序的 WMS 和 WMTS 服务、JavaScript 查看器和移动 SDK 的支持. （BSD)
    • ZeroNet - 使用比特币加密和 BitTorrent 网络的去中心化网站. （GNU GPLv2)
    • Simple Mobile Tools - 带有可自定义小部件的 Android 应用程序，没有烦人的广告和不必要的权限.
    • IP2Trace - 使用 IP2Location 数据库显示地理位置信息的跟踪路由工具. （MIT)

    Payments¶

    • Electrum - 轻量级比特币客户端，提供钱包恢复、去中心化服务器和离线存储. （MIT)
    • GNU Taler - 类似现金的在线支付系统. （GNU GPL)

    Scientific Work¶

    • DocEar - 使用思维导图技术进行文献管理. （GNU GPLv2+)
    • JabRef - 使用 BibTeX/BibLaTeX 管理参考文献和附加的 PDF. 包括对 LibreOffice 的支持. （MIT)

    Screencasting¶

    • OBS Studio - 适用于所有平台的屏幕录像机和视频流应用程序. （GNU GPLv2)
    • ShareX - 屏幕截图、文件共享和生产力工具. （GNU GPLv3)
    • SimpleScreenRecorder - 高度可定制的 GUI 截屏程序. （GNU GPLv3)

    Text Editors¶

    • Atom - 可破解的文本和源代码编辑器. （已存档）（MIT)
    • Pulsar - 社区主导的超级可破解文本编辑器. 从 Atom 分叉. （MIT)
    • Emacs - 可扩展、可定制、免费/自由的文本编辑器 - 等等. （GNU GPL)
    • GNU nano - 命令行的简单文本编辑器. （GNU GPL)
    • vim - Vim 是一个高度可配置的文本编辑器，旨在非常高效地创建和更改任何类型的文本. （GNU GPL compatible)
    • VS Code - Microsoft 制作的源代码编辑器（MIT)
    • Notepad++ - 可扩展的文本和源代码编辑器. （GNU GPLv2+)

    Utilities¶

    • Grayscale Mode - 一款开源 macOS 应用程序，可让您直接从菜单栏或使用键盘快捷键 (⌥⌘G) 快速切换灰度滤镜. （MIT)
    • Markdownify - 一个最小的 Markdown 编辑器. （MIT)
    • Sandman - 让您知道何时关闭计算机并根据计算的睡眠周期进行睡眠. （MIT)
    • Search Deflector - 一个小程序，可将从 Windows 开始菜单或 Cortana 进行的搜索重定向到您喜欢的任何浏览器和搜索引擎. （MIT)
    • Wireshark - 用于网络故障排除和分析的数据包分析器. （GNU GPLv2)
    • Espanso - 跨游戏形式的文本扩展器（GNU GPLv3)

    Video¶

    • NewPipe - 适用于 Android 的轻量级 YouTube 前端. （GNU GPLv3)
    • Kodi - 家庭影院软件和娱乐中心. （GNU GPLv2)
    • VLC - 支持多种音频和视频格式的多媒体播放器. （GNU GPLv3)
    • youtube-dl - 功能丰富的命令行程序，用于从 YouTube 和其他视频网站下载视频. （Unlicense)
    • MPV - 具有脚本支持的可定制多媒体播放器. （GNU GPLv2)

    Virtualization¶

    • QEMU - QEMU 是一个通用的开源机器模拟器和虚拟器. （GNU GPLv2)
    • VirtualBox - 功能丰富的虚拟化工具，支持多种操作系统. （GNU GPLv2)

    Web Applications¶

    • Etherpad - 实时协作文档编辑. （Apache License 2.0)
    • Ghost - 用于构建和运行在线出版物的可破解平台. （MIT)
    • GitLab - 整个代码生命周期的 Git 存储库管理器. （MIT)
    • GNU social - 用PHP编写的微博服务器. （GNU GPLv3)
    • Gogs - 轻量级、跨平台、自托管的 Git 服务. （MIT)
    • Healthchecks - Cron 作业监控服务. （BSD 3-clause)
    • Inventaire - 与朋友和社区分享书籍. （GNU AGPLv3)
    • Lobsters - 链接聚合和讨论以及否决解释. （BSD 3-clause)
    • Mastodon - 去中心化的社交网络服务器. （GNU AGPLv3)
    • MediaGoblin - 各类媒体的发布平台. （GNU AGPLv3)
