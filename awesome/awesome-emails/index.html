    很棒的电子邮件 ✉ ✨

    一系列很棒的框架、模板、工具、工作流程等，可用于构建更好的电子邮件.

    Frameworks¶

    React¶

    • React Email - 一系列高质量、无样式的组件，用于使用 React 和 TypeScript 创建精美的电子邮件.
    • Mailing - 在 React 中构建电子邮件，从任何地方发送.
    • JSX-email - JSX Mail 是一个使用 React 创建电子邮件模板的电子邮件框架.

    Misc¶

    • MJML - MJML 是一种标记语言，旨在减少编写响应式电子邮件的难度.
    • Foundation for Emails - 快速创建适用于任何设备和客户端的响应式 HTML 电子邮件. 甚至展望.
    • Email Framework - HTML 电子邮件框架旨在帮助您构建响应式 HTML 电子邮件.
    • HEML - HEML 是一种用于构建响应式电子邮件的开源标记语言.
    • Acorn - 基于黄金比例排版网格的响应式电子邮件框架.
    • Pine - 12 列网格、响应式电子邮件框架.
    • Maizzle - 使用 Tailwind CSS 和高级的电子邮件特定后处理快速构建 HTML 电子邮件原型.

    Cheatsheets¶

    • Rico's cheatsheet - Rico 的 HTML 电子邮件备忘单.

    Templates¶

    • Cerberusemail - 响应式 HTML 电子邮件模板的模式.
    • Responsive templates - 来自 EmailOctopus 的免费 HTML 模板，可与任何提供商一起使用.
    • Campaign - 轻松编写响应式电子邮件模板，用模型填充它们，然后发送出去.
    • Sendwithus - 开源电子邮件模板.
    • Mailchimp Email Blueprints - 电子邮件蓝图是 HTML 电子邮件模板的集合，可以作为电子邮件设计的坚实基础和起点.
    • Campaign Monitor Templates - 轻松创建精美的品牌电子邮件营销活动.
    • Responsive Transactional HTML Email Templates - 每个模板都是响应式的，并且每个模板都已经在所有流行的电子邮件客户端中进行了测试.
    • Cerberus - 响应式电子邮件模式很少.
    • Antwort - Antwort 为电子邮件提供响应式布局，既适合又适应客户端宽度.
    • Responsive HTML Template - 该模板设计用于所有主要电子邮件平台，并使用媒体查询构建，使其能够响应支持媒体查询的智能手机.
    • HTML Email Templates - 适用于初创公司和 SaaS 的响应式 HTML 电子邮件模板.
    • Free Responsive template : Slate - 五个预先测试的模板，适用于任何邮寄服务.
    • Grunt Email Boilerplate - 用于创建电子邮件的 Grunt 模板.
    • Responsive Email Patterns - 响应式电子邮件的模式和模块的集合.
    • Email boilerplate - 用于发送格式良好的消息的电子邮件样板.
    • htmlemail - 适合初创公司和开发人员的响应式 HTML 电子邮件模板.
    • Postmark Transactional Email Templates - 坚如磐石的应用程序电子邮件模板.
    • Stripo - Stripo 提供丰富的响应式、交互式精美电子邮件模板.
    • Postcards - 在线设计响应式电子邮件模板，无需编码知识.
    • Lee Munroe's Grunt Email Design - 用于使用 SCSS 设计和测试响应式 HTML 电子邮件模板的 Grunt 工作流程.

    Tools¶

    • Can I Email - 我可以通过电子邮件发送... 支持电子邮件中的 HTML 和 CSS 表格.
    • Topol - 一分钟内创建令人惊叹的响应式 HTML 电子邮件.
    • MJML Desktop App - MJML 官方桌面应用程序.
    • Rollover - 生成翻转图像的代码，当电子邮件收件人将光标悬停在图像上时，该代码会显示替代图像.
    • Bulletproof Email Buttons - 使用逐步增强的 VML 和 CSS 设计华丽的按钮.
    • Bulletproof Background Images - 在 VML 和 CSS 的帮助下为您的 HTML 电子邮件创建坚如磐石的背景图像.
    • Stripo.email - 在线拖放和 HTML 电子邮件生成器，无需任何 HTML 技能即可快速创建专业且响应迅速的电子邮件模板.
    • MySigMail Card - 免费的拖放生成器可帮助您直观且快速地创建电子邮件模板.
    • TestSubject - 测试您的电子邮件主题行在最流行的移动设备上的显示方式.
    • MailSpice analytics - 适用于所有电子邮件服务的电子邮件分析插件，可深入了解您的所有电子邮件活动（适用于所有 ESP）.
    • Alter.email - 一款免费工具，可帮助您内联 CSS、删除未使用的 CSS、清理文本、向标签添加额外属性、添加 URL 跟踪参数、缩小甚至美化 HTML 电子邮件.
    • Sensitive Subjects - 切勿发送令人尴尬的主题行！
    • Detergent - 准备文本以粘贴到 HTML 中的工具：删除不可见的 Unicode 字符（例如 ETX 它可以来自 PSD 文件），对特殊字符进行编码（例如将 £ 转换为 £）并改进排版.
    • EmailComb - 电子邮件友好的未使用 CSS 删除工具，它不会解析，因此接受混合源，包括 ESP 模板标签. 附加功能：CSS 丑化和 HTML/CSS 缩小（由 HTML Crush 提供支持）.
    • HTML Crush - 电子邮件友好的 HTML 缩小器：将行限制为 500 个字符. （最佳实践），尊重 Outlook 条件标记并接受混合源，包括 ESP 模板标记. 或者，只能删除缩进（不触及换行符）.
    • Tabifier - 与 HTML Crush 相反 - 一个美化 HTML 文件缩进的工具. 适用于混合输入源，包括 ESP 模板标签.
    • Email Link Builder - 制作 mailto 链接，可打开包含预填收件人、主题、正文等内容的电子邮件草稿.
    • email-verifier - 无需发送任何电子邮件即可进行电子邮件验证的工具.
    • DevMail - 面向软件开发人员的电子邮件沙箱. 从您的开发、暂存、QA 或 CI 环境捕获电子邮件.
    • Disposable Email Check API - 一个免费的 API，用于检查电子邮件地址是否是一次性的. 无需注册.

    • PutsMail - 在发送 HTML 电子邮件之前测试您的 HTML 电子邮件的免费工具！
    • Litmus - 使用 Litmus 测试和跟踪您的电子邮件.

    • Grunt Litmus - 使用 grunt 向 Litmus 发送电子邮件测试.
    • Gulp Litmus - 使用 gulp 向 Litmus 发送电子邮件测试.
    • MailTrap - 一个假 SMTP 服务器，用于测试、查看和共享电子邮件，而不会向真实客户发送垃圾邮件.
    • testmail.app - 具有无限邮箱的假 SMTP 服务器和用于自动化端到端测试的 GraphQL API.
    • MailHog 本地假 SMTP 服务器（基于 Go）.
    • MailCatcher 本地假 SMTP 服务器（基于 Ruby）.
    • Email Privacy Tester - 测试电子邮件客户端隐私功能（加载远程内容）的工具.
    • Mail Tester - 测试您的电子邮件是否属于垃圾邮件.
    • Sendtest.email - 发送前测试您的 HTML 电子邮件、⚡4email 或文本版本. 注重隐私，无需注册！
    • Mailsnag - 测试电子邮件服务器以满足您的所有 SMTP 需求 - 模拟延迟和错误、预览和共享电子邮件等等 - 所有这些都无需向实际客户发送电子邮件.
    • Email Spam Tester - 免费工具可检查您的电子邮件是否会进入垃圾邮件文件夹.
