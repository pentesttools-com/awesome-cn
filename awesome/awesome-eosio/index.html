    Awesome EOSIO ¶

    > 为用户和开发人员精选的 EOSIO 资源列表.

    EOSIO 是去中心化应用程序和开源智能合约平台最强大的基础设施. EOSIO 软件使企业能够快速构建和部署高性能和高安全性的基于区块链的应用程序.

    欢迎投稿. 通过拉取请求添加链接或创建问题以开始讨论. 请先阅读贡献指南.

    Services¶

    Developer Tools¶

    • EOSIO/eosio.cdt - EOSIO.CDT（合约开发工具包）是一套用于构建 EOSIO 合约的工具.
    • Blanc - 用于构建基于 WebAssembly 的区块链合约的工具链. 目前，Blanc 支持 EOSIO 和 CosmWasm（实验性）.
    • EOSIO/ricardian-template-toolkit - Ricardian 合约规范的渲染器.
    • EOSIO Ethereum Virtual Machine (EVM) - EOSIO 上的以太坊虚拟机.
    • Elliptic curve cryptography functions (ECC) - 私钥、公钥、签名、AES、加密/解密.
    • Securely Generating an EOS Key Pair Using Official EOS.IO Code - YouTube.
    • EOSEssentials/EOSDrops - EOSIO 区块链的空投工具.
    • Octopus - WebAssembly 和区块链智能合约 (BTC/ETH/NEO/EOS) 的安全分析工具.
    • greymass/eosio-resources - 协助网络资源计算的工具.
    • greymass/abi2core - 用于将 ABI 定义转换为 greymass/eosio 类型定义的命令行实用程序.
    • maxifom/eos-abigen - 用于为 EOS 兼容合约生成类型安全客户端和表的 Typescript 和 Golang 代码生成器.
    • manh-vv/eosjs-name - 将 Eosio 名称转换为 uint64 并返回.
    • blockchain-etl/eos-etl - 谷歌 BigQuery 公共数据集的连接器.

    Code examples¶
