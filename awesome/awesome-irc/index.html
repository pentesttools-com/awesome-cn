    Awesome irc

    Awesome IRC ¶

    > 精选清单 IRC 资源.

    与 Internet 中继聊天 (IRC) 协议相关的工具、软件和其他资源的列表.

    IRC（互联网中继聊天）是一种开源协议，可用于通过渠道进行基于多用户文本的通信.

    Clients¶

    您使用这些连接到 IRC.

    • Textual - 高度可定制、ZNC 集成、iCloud 同步（4.99 美元）. (source)macOS
    • LimeChat - 多个服务器的一个窗口，键盘快捷键，快速稳定. (source) macOS iOS
    • HexChat - 基于 XChat，易于使用，拼写检查和多语言. (source) Windows macOS Linux
    • gamja - 一个简单的 IRC 网络客户端. (source）网络
    • Kiwi IRC - 强大的现代网络 IRC 信使. (source, demo）网络
    • CIRC - 使用 chrome.sockets API 直接连接到 IRC 服务器，无需代理. (source）铬
    • Quassel - 分布式（客户端可以连接到永久在线的中央核心或从中分离.（source) Linux macOS Windows
    • Yaaic - 多服务器/频道支持、SASL 支持、平滑的频道滚动/滑动. (source)安卓
    • relay.js - 专注于使 IRC 不那么令人生畏和更易于使用. 网络
    • Circe - 在 Emacs 中使用，合理的默认值. Emacs
    • Smuxi - 用户友好，基于 GNOME / GTK+. (source) Linux Windows macOS
    • KvIRC - 免费、便携、基于 Qt GUI 工具包. (source) Linux macOS Windows
    • Konversation - 建立在 KDE 平台上的用户友好的客户端. (source)Linux
    • sic - Simple IRC client - 不到 250 行 C 的终端客户端.Linux
    • irssi - 终端客户端，模块作者的多协议友好，GPLv2. Linux macOS Cygwin BSD
    • Revolution IRC - 功能齐全、积极维护的 Android IRC 客户端. 安卓
    • AdiIRC - 从来没有客户为 IRC 体验的各个方面提供如此精细的设置. (features, screenshots) Windows WINE
    • IRC for Android™ - 适用于高级用户的 Android/Chrome 操作系统客户端，具有 ZNC 内置、通知逻辑、可靠的 DCC、硬件键盘的键绑定等.Android ChromeOS
    • Iridium - 内置 Vala 和 GTK 的友好 IRC 客户端，专为 elementary OS 设计. (source)Linux
    • IRC Vitamin - 简单、快速、轻松地访问多个 IRC 网络. 安卓

    更多的？ 找到包括保镖的客户 below.

    Bouncers¶

    对于断开连接和重新连接而不丢失聊天会话很有用.

    Hosted¶

    • IRCCloud - 团队、朋友和社区的群聊. 保持联系，随时随地聊天，绝不会错过任何消息（+客户）（0-3.50 英镑/月）.
    • iOS App - 官方的. 目标-C
    • Android App - 官方的. 爪哇
    • Nimbus - 独立客户端. macOS Objective-C

    Self-hosted¶

    • Convos - 始终在线的网络 IRC 客户端. (source) Perl JavaScript Web
    • ZNC - 最受欢迎. 许多不同的插件. (source)C++
    • BIP IRC Proxy - 始终在线，轻量级和安全的开源 IRC 代理与积压. (source)C
    • TheLounge - 响应式、自托管和支持多个用户. (source, demo) JavaScript Node.js Web
    • WeeChat - 快速、轻便且可扩展的聊天客户端. (source) Linux macOS Windows
    • soju - 一个用户友好的 IRC 保镖. 去

    Daemons¶

    用于运行您自己的 IRC 服务器或网络.

    • ircd.js - 服务器将允许客户端连接、加入频道、更改主题； 基本的东西.
    • InspIRCd - 模块化，稳定，从头开始编写. (source)
    • miniircd - 非常简单和有限.
    • ngIRCd - 适用于小型或专用网络的便携和轻量级. (source)
    • Ergo - 可移植且围绕规范设计的现代服务器（前沿的 IRCv3 支持）.
    • RobustIRC - 没有 netsplits 的 IRC 服务器. (source)

    Services¶

    用于向您的网络提供用户帐户和机器人程序，例如 NickServ/ChanServ.

    • Atheme - 专为具有高可扩展性要求的大型网络而设计. (source)
    • anope - 专为灵活性和易用性而设计. (source)

    Bots¶

    为人类提供服务的 IRC 用户，例如集成或信息.

    • Sopel - 成吨的现成功能、教程、完整记录. (source)蟒蛇
    • Limnoria - 健壮，用户友好，开发人员友好. 蟒蛇
    • Twitch Plays - 从聊天中获取输入并按下相应的键. 蟒蛇
    • Skybot - 主要目标是简单和强大. 蟒蛇
    • talkbackbot - 使用著名女性的名言来响应配置的触发短语. (source)蟒蛇
    • lazybot - 用户友好且功能强大. Clojure
    • IRC-BF - 脑残
    • geordi - 编译并运行 C++ 代码片段. C++
    • CloudBot - 简单、快速、可扩展. 蟒蛇
    • yossarian-bot - 大型默认插件集，基于 Cinch. 红宝石
    • helga - 支持多种协议的可插拔聊天机器人. 蟒蛇
    • EveIRC - 可扩展的聊天/频道/服务器管理服务提供机器人. 使用 Cinch Framework . 红宝石
    • BitBot - Modular, event-driven bot featuring a REST API, individual user settings and much more. (bitbot.dev)蟒蛇
    • Cardinal - 专注于简化插件开发的 Python Twisted IRC 机器人. 蟒蛇
    • pyHoneybot - 专注于简化插件开发的 Python Twisted IRC 机器人. (source)蟒蛇
    • wayback - 带有 IRC 接口的归档工具，集成了各种归档服务.

    Frameworks¶

    有助于编写机器人或将 IRC 与应用程序集成.

    • node-irc JavaScript
    • goirc - 基于事件，有状态，缺乏文档. 去
    • Hubot IRC Adapter - hubot 的 IRC 适配器. JavaScript
    • go-ircevent - 基于事件. 去
    • slate-irc - 插件系统、简单的 api、任意输入流、调试支持. JavaScript
    • PircBotX - 基于事件的 IRC 库，具有简单的 API（更新的分支 PircBot ). 爪哇
    • IRC::Client - 基于 Perl6 的可扩展 IRC 客户端框架.
    • irccd - 可使用 JavaScript 自定义的灵活 IRC 机器人. C++.

    • Biboumi - XMPP 客户端 C 的 IRC 网关
    • teleirc - 电报 ↔ IRC. JavaScript
    • toxirc -毒物 ↔ IRC. C
    • skyweb2irc - Skype（网络客户端 API）↔ IRC. Javascript
    • matterbridge - IRC ↔ Mattermost ↔ Discord ↔ XMPP ↔ Gitter ↔ Slack ↔ Discord ↔ Telegram ↔ etc.Go
    • Heisenbridge - 保镖式 Matrix IRC 桥Python
