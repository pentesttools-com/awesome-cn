    新手友好项目

    Awesome First PR Opportunities ¶

    灵感来自 First Timers Only 博客文章.

    如果您是开源项目的维护者，请将标签“first-timers-only”（或类似标签）添加到您的项目并在此处列出，以便人们可以找到它.

    如果您不是程序员但想做出贡献，请查看 Awesome for non-programmers 列表.

    .NET¶

    • Legerity （标签：好第一期）
      一个框架，用于使用 .NET 上的 Appium/Selenium 加速针对 Windows、Android、iOS 和 Web 的自动化 UI 测试的开发.
    • Legerity for Uno Platform （标签：好第一期）
      Legerity 的扩展框架，用于加速在 .NET 上使用 Appium/Selenium 为 Uno Platform 应用程序开发自动化 UI 测试.
    • MvvmCross （标签：仅限初学者）
      适用于跨平台解决方案的 .NET MVVM 框架，包括 Xamarin.iOS、Xamarin.Android、Windows 和 Mac.
    • RawCMS （标签：好第一期）
      RawCMS 是用 ASP.NET Core 编写的无头 CMS，专为采用 API 优先技术的开发人员而构建.
    • Shouldly （标签：跳入）
      应该测试 .NET - 断言的方式应该是！

    Ansible¶

    • Exosphere （标签：好首期）
      Exosphere 是基于 OpenStack 的云系统的用户友好客户端界面.

    C¶

    • Neovim （标签：好第一期）
      Vim-fork 专注于可扩展性和敏捷性.
    • Profanity （标签：好第一期）
      Ncurses XMPP 聊天客户端.

    C¶

    • Cake (label: Good-first-issue)
      Cake (C# Make) is a free and open source cross-platform build automation system with a C# DSL for tasks such as compiling code, copying files and folders, running unit tests, compressing files and building NuGet packages.
    • grok.net （标签：好第一期）
      跨平台 .NET grok 实现
    • OpenRA （标签：简单）
      一个实时战略游戏引擎，支持早期的 Westwood 经典游戏，例如命令与征服：红色警戒.
    • osu! （标签：好第一期）
      音乐游戏. 节奏只需点击一下！
    • Uno Platform (label: good-first-issue)
      OSS project for creating pixel-perfect, single-source C# and XAML apps which run natively on iOS, Android, macOS, Linux and Web via WebAssembly.

    C++¶

    • electron （标签：好第一期）
      使用 JavaScript、HTML 和 CSS 构建跨平台桌面应用程序
    • Godot Engine (label: good first issue)
      2D and 3D cross-platform game engine. Also has C# and Python code.
    • MoveIt （标签：好第一期）
      易于使用的开源机器人操作平台，用于开发商业应用程序、原型设计和基准测试算法.
    • projectM （标签：好第一期）
      使用 OpenGL 和 GLSL 的音乐可视化库. 有使用 Qt5、SDL、emscripten、iTunes、Kodi 的应用程序.
    • Roc Toolkit （标签：求助）
      用于通过网络实时传输音频的工具包.
    • tensorflow （标签：统计：贡献-欢迎）
      使用数据流图进行可扩展机器学习的计算
    • Yugabyte DB （标签：好第一期）
      分布式 SQL 数据库.

    Clojure¶

    • Alda （标签：唾手可得的果实）
      一种音乐家的音乐编程语言.

    ClojureScript¶

    • LightTable （标签：好第一期）
      下一代代码编辑器！ KickStarter 上资金最多的项目之一.

    Dart¶

    • dart.dev （标签：初学者）
      一个涵盖 Dart 语言和公共库的网站，适用于 Dart 库、Web 应用程序、服务器端代码和移动 (Flutter) 应用程序的开发人员.
    • flutter （标签：好的第一次贡献）
      Flutter 是 Google 的 UI 工具包，用于从单个代码库为移动、网络、桌面和嵌入式设备构建漂亮的、本地编译的应用程序.

    Elixir¶

    • Ecto （标签：级别：初学者）
      Ecto 是 Elixir 的数据库包装器和语言集成查询
    • Elixir （标签：级别：初学者）
      Elixir 是一种动态的函数式语言，旨在构建可扩展和可维护的应用程序

    Elm¶

    • Exosphere （标签：好首期）
      Exosphere 是基于 OpenStack 的云系统的用户友好客户端界面.

    Go¶

    • containerd （标签：exp/初学者）
      强调简单性、稳健性和可移植性的行业标准容器运行时.
    • Docker/CLI （标签：exp/初学者）
      Docker CLI
    • Helm （标签：好第一期）
      Kubernetes 包管理器
    • httpexpect （标签：求助）
      Go 的端到端 HTTP 和 REST API 测试.
    • Hugo （标签：GoodFirstIssue）
      用 GoLang 构建的快速灵活的静态站点生成器
    • Killgrave （标签：好第一期）
      在 Go 中生成模拟服务器的简单方法.
    • Kubernetes （标签：好第一期）
      生产级集装箱调度和管理系统
    • lxd （标签：简单）
      系统容器和虚拟机管理器.
    • Mattermost （标签：不适用）
      Golang 和 React 中的开源 Slack 替代方案
    • Meshery （标签：好第一期）
      Meshery，服务网格管理平面.
    • Moby （标签：exp/初学者）
      开源应用容器引擎
    • PureLB （标签：不适用）
      使用标准 Linux 网络和路由协议的 Kubernetes 负载平衡器协调器.
    • script （标签：好第一期）
      一个 Go 库，用于执行 shell 脚本擅长的任务：读取文件、执行子进程、计算行数、匹配字符串等. 非常欢迎初学者，他们将在 PR 过程中获得详细的代码审查和帮助.
    • Terraform （标签：好第一期）
      一种用于安全高效地构建、更改和版本控制基础架构的工具.
    • TiDB （标签：新贡献者）
      分布式可扩展混合事务和分析处理 (HTAP) 数据库

    Haskell¶

    • Hasura GraphQL Engine （标签：好第一期）
      具有细粒度访问控制的 Postgres 上快速、即时的实时 GraphQL API，还可以触发数据库事件的 webhook.

    Java¶

    • appsmith （标签：好第一期）
      拖放内部工具生成器
    • Codename One （标签：好第一期）
      面向 Java 开发人员的跨平台移动应用程序开发框架
    • elasticsearch （标签：好第一期）
      开源、分布式、RESTful 搜索引擎.
    • Images-to-PDF （标签：好第一期）
      将图像转换为 PDF 文件的 Android 应用程序.
    • JabRef （标签：好第一期）
      使用现代 Java 功能（包括 JavaFX）管理文献参考的桌面应用程序. 致力于代码质量和建设性反馈：每个 Pull Request 都由两名开发人员审查，以提供高质量的反馈并确保新贡献的高质量.
    • OpenMetadata （标签：好第一期）
      OpenMetadata 是一个用于数据发现、数据质量、可观察性、治理、数据沿袭和团队协作的一体化平台.
    • SirixDB （标签：好第一期）
      SirixDB 是一种进化的、版本化的 NoSQL 文档存储（XML 和 JSON）（主要）用 Java 编写. 它使用从 ZFS 和 Git 借来的许多概念在提交期间存储紧凑的快照. 每个修订都被索引，并且可以使用临时查询来查询文档存储. 它特别适合现代硬件.
    • Strongbox （标签：好第一期）
      Strongbox 是一个用 Java 编写的工件存储库管理器.
    • TEAMMATES （标签：好第一期）
      TEAMMATES 是一个免费的在线工具，用于管理学生的同伴评价和其他反馈途径.
    • Trino (formerly Presto SQL) (label: good-first-issue)
      A distributed SQL query engine for big data. Ask for guidance on project's Slack.
    • Wikimedia Commons Android App （标签：好第一期）
      允许用户将图片从他们的 Android 手机/平板电脑上传到维基共享资源.
    • XWiki (label: n/a)
      XWiki is a free wiki software platform written in Java with a design emphasis on extensibility. Beginners should follow the onboarding wiki.
    • zerocode （标签：好第一期）
      无需编码的 API 自动化、简单的 JSON 响应断言、测试 REST、SOAP、Kafka 和 Java/DB API、CI/Jenkins 友好.

    JavaScript¶

    • altair (label: good-first-issue)
      A beautiful feature-rich GraphQL Client for all platforms.
    • Ancient Beast （标签：简单）
      回合制策略游戏，您可以 3D 打印一队具有独特能力的生物来击败敌人.
    • appsmith （标签：好第一期）
      拖放内部工具生成器
    • AVA （标签：适合初学者）
      未来派测试赛跑者.
    • Babel （标签：好第一期）
      用于编写下一代 JavaScript 的编译器.
    • Binari （标签：待售）
      具有实时二叉树视觉效果的交互式代码编辑器，旨在向新开发人员传授动态编程的基础知识.
    • Botpress （标签：贡献者友好）
      构建优秀机器人的唯一明智方法.
    • Brave Browser （标签：好第一期）
      适用于 macOS、Windows 和 Linux 的桌面浏览器.
    • cdnjs （标签：好第一期）
      最好的源码Web前端资源CDN
    • Check It Out （标签：好第一期）
      Check It Out 是一个类似于 ncurses 的 CLI，让用户可以交互地导航并选择一个 git 分支来签出.
    • Create React App （标签：好第一期）
      创建没有构建配置的 React 应用程序.
    • cypress （标签：好第一期）
      对在浏览器中运行的任何内容进行快速、简单和可靠的测试.
    • electron （标签：好第一期）
      使用 JavaScript、HTML 和 CSS 构建跨平台桌面应用程序
    • Ember.js （标签：适合新贡献者）
      用于创建雄心勃勃的 Web 应用程序的 JavaScript 框架.
    • Ember.js Data （标签：适合新贡献者）
      Ember.js 的数据持久性库.
    • ESLint （标签：好第一期）
      一个完全可插入的工具，用于识别和报告 JavaScript 中的模式.
    • eslint-plugin-unicorn （标签：适合初学者）
      很棒的 ESLint 规则.
    • Fastify (label: good-first-issue)
      Fast and low overhead web framework, for Node.js.
    • freeCodeCamp （标签：仅限初学者）
      开源代码库和课程. 学习编码并帮助非营利组织.
    • Gatsby.js （标签：好第一期）
      使用 React 构建快速、现代的应用程序和网站.
    • Ghost （标签：好第一期）
      只是一个博客平台
    • Habitica （标签：好第一期）
      Habitica 是一个游戏化的任务管理器、web 应用程序和 android/ios 应用程序，非常棒的氛围. 此处贡献指南（mongo、express、vue、webapp 的节点堆栈）
    • Hoppscotch （标签：好第一期）
      免费、快速且美观的 API 请求生成器.
    • Hyper （标签：好第一期）
